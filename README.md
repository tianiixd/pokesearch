# PokeSearch

PokeSearch is a web application that allows users to search for information about Pokémon. It provides details such as Pokémon types. The application is built using React and utilizes the PokéAPI to fetch data.

## Features

- Search for Pokémon by name or ID.
- Results display the Pokémon's name, image, types, and other relevant information.
- Responsive design for optimal viewing on various devices.

## Technologies Used

- React: A JavaScript library for building user interfaces.
- PokéAPI: A RESTful API that provides data about Pokémon.
- Tailwindcss: For styling the application.
